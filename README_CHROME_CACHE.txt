Chrome eski CSS/JS dosyalarini cache'ten okuyabilir.

Cozum:
1. Bu klasoru acin.
2. Chrome'da sayfayi acin.
3. Ctrl + F5 veya Ctrl + Shift + R ile hard reload yapin.
4. Hala eski gorunurse:
   Chrome > DevTools > Network > Disable cache isaretleyin ve sayfayi yenileyin.
